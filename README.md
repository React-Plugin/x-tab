# x-tabs
# 安装
```
npm install --save xui-tabs
```
# 使用
```
<Tabs defaultActive="1" onChange={this.onChange}>
    <TabPane tab="选项卡一" key="1">这是选项卡一的内容</TabPane>
    <TabPane tab="选项卡二" key="2">这是选项卡2的内容</TabPane>
</Tabs>
<Tabs active={this.state.index} onChange={this.onChange}>
    <TabPane tab="选项卡一" key="1">这是选项卡一的内容</TabPane>
    <TabPane tab="选项卡二" key="2">这是选项卡2的内容</TabPane>
</Tabs>
```
